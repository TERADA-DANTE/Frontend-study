# Native object vs Host object

## Native Object

ECMAScript 明細で意味論的な部分を完全に定義した Object であり、次のようなものがある。

-   `Object`
-   `Function`
-   `Date`
-   `Math`
-   `parseInt`
-   `eval` ...

<br>

## Host Object

Javascript を実行する環境に属するものでその環境にしかない Object。Browser の場合次のようなものがある。

-   `window`
-   `document`
-   `location`
-   `XMLHttpRequest`
-   `querySelectorAll` ...

<br>
